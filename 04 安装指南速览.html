§ 03 安装与启动
①
右键点击 02右键打开安装.command → 选「打开」
必须右键→打开，不能直接双击。macOS 对外部脚本的安全机制，首次运行必须这样操作，之后才能直接双击。
②
脚本自动安装 Git · Node.js · Claude Code
若弹出 Git 安装窗口，点「安装」，装完后按回车继续。API 配置自动写入，无需手动填写。
③
提示 Password: 时输入 Mac 开机密码
输入时屏幕不显示字符，这是正常的，输完直接回车。注意是开机密码，不是 Apple ID。
④
出现「安装成功」→ 关闭终端 → 重新打开终端 → 输入 claude 回车
安装后必须关掉旧终端，新打开的终端才能识别 claude 命令。
§ 04 操作授权说明

常用操作（读写文件、运行脚本等）已预置授权，无需每次手动同意。遇到未预置的操作会弹出询问：

选项 含义 建议
Allow 允许这一次，下次还会再问 刚开始用时推荐，每次都看清楚它要做什么
Allow always 允许这次，以后不再询问 熟悉后，对信任的操作选这个
Deny 拒绝这次操作 不确定时随时可以拒绝
§ 05 练手任务（示例项目）

把 03 示例项目 文件夹拖入终端窗口，路径自动出现，再依次输入以下提示词：

任务 01 · 摸清文件夹内容
› 帮我看看这个文件夹里有什么，每个文件大概是什么内容，整理成一份简要说明。
Claude 读取所有文件，输出每个文件的简要说明
任务 02 · 制作可视化分析网页
› 其中有两个素材文件，帮我做一个好看的 HTML 网页，展示用户反馈的分类汇总、高频问题排名、以及需求和用户痛点的匹配分析。直接生成本地 HTML 文件，生成后帮我用浏览器打开。
Claude 分析素材，生成 HTML 可视化文件，并在浏览器中打开
§ 06 常用指令
/clear
清空对话
话题切换、对话跑偏时，清掉全部重头开始
/resume
继续上次对话
弹出历史列表，选一条接着上次继续
/compact
压缩上下文
对话太长记性变差时，保留摘要压缩对话
/init
创建项目规则
扫描文件夹自动生成 CLAUDE.md 项目指令
/memory
查看长期记忆
跨会话记住的信息，可查看和管理
/help
查看所有指令
当前版本全部可用指令的完整列表
§ 07 如何配置
没有图形设置界面，通过文件实现配置。文件夹默认隐藏，按 ⌘ ⇧ . 显示/隐藏。
~/.claude/
├── CLAUDE.md  ← 全局指令
├── settings.json ← 技术设置
└── memory/     ← 长期记忆
~/.claude/CLAUDE.md
全局指令 · 最重要
对你打开的所有项目都生效。适合放普适规则，如：所有回答用中文、解释技术内容用通俗语言、修改文件前先列出改动计划。安装脚本已预置一份。
工作文件夹/CLAUDE.md
项目指令 · 按需设置
只对当前文件夹生效。适合放项目专属背景，如：这个文件夹是整理用户反馈的、不要动「已归档」目录。Claude 会同时读取全局指令和项目指令。
如何修改全局指令
启动 Claude Code 后直接说：

「帮我打开全局配置文件，我想加几条指令」

它会直接帮你写进去，不需要手动找文件。
§ 08 遇到问题？
双击 .command 文件没有反应
右键点击文件 → 选「打开」（不是双击）
弹窗提示「无法验证开发者」
点「取消」→ 系统设置 → 隐私与安全性 → 找到相关提示 → 点「仍要打开」
安装中途卡住不动了
等待 5 分钟无反应，关闭终端后重新右键运行安装脚本，已完成的步骤会自动跳过。
输入 claude 后没有反应
确认已关闭旧终端窗口，重新打开新的终端再试。安装完成后必须开新终端才能识别命令。
