# Claude Code 一键安装包（MiniMax 版）

面向非开发人员的 Claude Code 安装方案。使用国内可直接访问的 MiniMax API，全程自动化，不需要 VPN、不需要信用卡。

---

## 这是什么

Claude Code 是 Anthropic 官方出品的命令行 AI 助理，能直接读写你电脑上的文件——用一句自然语言，它就能帮你整理数据、写代码、生成报告，全程无需复制粘贴。

本安装包解决了国内用户的两个痛点：

- **API 访问**：默认绑定 MiniMax 的兼容接口，国内网络直连，不需要代理
- **安装门槛**：一个脚本搞定 Git、Node.js、Claude Code、API 配置，无需任何命令行基础

## 安装步骤

**前提**：macOS 系统，有 MiniMax API Key（[前往 MiniMax 开放平台申请](https://platform.minimaxi.com/)）

**第一步**：用文本编辑器打开 `02右键打开安装.command`，将第 13 行的占位符替换为你的真实 API Key：

```
API_KEY="sk-****** 此处换成你的 API KEY ********"
```

**第二步**：右键点击 `02右键打开安装.command` → 选「打开」（首次运行必须右键打开，这是 macOS 的安全机制）

**第三步**：脚本自动完成以下步骤（约 3–10 分钟）：
- 检查并安装 Git
- 下载并安装 Node.js v20
- 安装 Claude Code
- 写入 MiniMax API 配置和全局规则

**第四步**：看到「安装成功」提示后，关闭终端窗口，重新打开一个新终端，输入 `claude` 回车即可开始使用。

> 详细步骤和截图说明见 `01 Claude Code 安装指南.html`。

## 安装后的配置文件位置

脚本会自动创建以下配置（无需手动操作）：

```
~/.claude/
├── CLAUDE.md        # 全局指令（预置：中文回答、通俗解释技术内容）
├── settings.json    # API 地址、密钥、预授权操作列表
└── memory/          # Claude 的长期记忆（运行后自动维护）
```

如需修改全局指令，启动 Claude Code 后直接说：「帮我打开全局配置文件，我想加几条规则」即可。

## 常见问题

**双击 `.command` 文件没有反应？**
必须右键 → 选「打开」，不能直接双击。首次运行 macOS 需要你手动授权。

**弹窗提示「无法验证开发者」？**
点「取消」关掉弹窗 → 打开「系统设置 → 隐私与安全性」→ 找到相关提示，点「仍要打开」。

**安装中途卡住？**
等待 5 分钟，若仍无反应，关闭终端后重新右键运行。已完成的步骤会自动跳过。

**输入 `claude` 后提示找不到命令？**
确认已关闭安装时的旧终端，重新打开一个新终端再试。
