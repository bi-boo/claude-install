# Claude Code 一键安装包（MiniMax 版 · Windows）

面向非开发人员的 Claude Code 安装方案。使用国内可直接访问的 MiniMax API，全程自动化，不需要 VPN、不需要信用卡。

---

## 这是什么

Claude Code 是 Anthropic 官方出品的命令行 AI 助理，能直接读写你电脑上的文件——用一句自然语言，它就能帮你整理数据、写代码、生成报告，全程无需复制粘贴。

本安装包解决了国内用户的两个痛点：

- **API 访问**：默认绑定 MiniMax 的兼容接口，国内网络直连，不需要代理
- **安装门槛**：一个脚本搞定 Git、Node.js、Claude Code、API 配置，无需任何命令行基础

---

## 系统要求

- **操作系统**：Windows 10（版本 1803 或更高）或 Windows 11
- **架构**：64 位系统
- **网络**：需要联网下载安装包
- **MiniMax API Key**：[前往 MiniMax 开放平台申请](https://platform.minimaxi.com/)

## 安装步骤

**第一步**：用记事本打开 `02 双击打开安装.bat`，将第 23 行的占位符替换为你的真实 API Key：

```
set "API_KEY=sk-****** 此处换成你的 API KEY ********"
```

**第二步**：双击 `02 双击打开安装.bat` 运行
- 如果弹出「Windows 保护了你的电脑」蓝色窗口，点「更多信息」→「仍要运行」
- 如果弹出「用户账户控制」窗口（问你是否允许更改），点「是」

**第三步**：脚本自动完成以下步骤（约 3–10 分钟）：
- 检查并安装 Git
- 下载并安装 Node.js v20
- 安装 Claude Code
- 写入 MiniMax API 配置和全局规则

**第四步**：看到「安装成功」提示后，关闭当前窗口，打开 PowerShell（按 Win 键搜索「PowerShell」），输入 `claude` 回车即可开始使用。

> 详细步骤和截图说明见 `01 Claude Code 安装指南.html`。

## 安装后的配置文件位置

脚本会自动创建以下配置（无需手动操作）：

```
C:\Users\你的用户名\
├── .claude.json          # 跳过首次引导（安装脚本自动设置）
└── .claude\
    ├── CLAUDE.md         # 全局指令（预置：中文回答、通俗解释技术内容）
    ├── settings.json     # API 地址、密钥、预授权操作列表
    └── memory\           # Claude 的长期记忆（运行后自动维护）
```

如需修改全局指令，启动 Claude Code 后直接说：「帮我打开全局配置文件，我想加几条规则」即可。

## 常见问题

**双击 .bat 文件弹出蓝色「Windows 保护了你的电脑」？**
点「更多信息」→「仍要运行」。这是 Windows SmartScreen 对未签名脚本的安全提示。

**弹出「用户账户控制」窗口？**
点「是」。安装软件需要管理员权限，这是 Windows 正常的安全确认。

**安装中途卡住？**
等待 5 分钟，若仍无反应，关闭窗口后重新双击运行。已完成的步骤会自动跳过。

**输入 `claude` 后提示「不是内部或外部命令」？**
确认已关闭安装时的旧窗口，重新打开一个新的 PowerShell 再试。如果仍不行，重启电脑后再试。

**安装过程中报网络错误？**
脚本优先使用国内镜像下载，如果仍然失败，请确保电脑能正常上网（打开网页试试）。
