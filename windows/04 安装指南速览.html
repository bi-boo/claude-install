§ 03 安装与启动
①
用记事本打开 02 双击打开安装.bat，填入你的 API Key
右键文件 → 选「编辑」或「用记事本打开」，找到第 23 行的占位符，替换为你的 MiniMax API Key，保存后关闭。
②
双击 02 双击打开安装.bat 运行安装
如果弹出蓝色「Windows 保护了你的电脑」，点「更多信息」→「仍要运行」。弹出权限确认窗口点「是」。
③
脚本自动安装 Git · Node.js · Claude Code
全程自动，无需任何操作。下载和安装过程约 3-10 分钟，取决于网速。API 配置自动写入。
④
出现「安装成功」→ 关闭窗口 → 打开 PowerShell → 输入 claude 回车
按 Win 键搜索「PowerShell」打开。或在桌面空白处右键 →「在终端中打开」（Windows 11）。
§ 04 操作授权说明

常用操作（读写文件、运行脚本等）已预置授权，无需每次手动同意。遇到未预置的操作会弹出询问：

选项 含义 建议
Allow 允许这一次，下次还会再问 刚开始用时推荐，每次都看清楚它要做什么
Allow always 允许这次，以后不再询问 熟悉后，对信任的操作选这个
Deny 拒绝这次操作 不确定时随时可以拒绝

§ 06 常用指令
/clear
清空对话
话题切换、对话跑偏时，清掉全部重头开始
/resume
继续上次对话
弹出历史列表，选一条接着上次继续
/compact
压缩上下文
对话太长记性变差时，保留摘要压缩对话
/init
创建项目规则
扫描文件夹自动生成 CLAUDE.md 项目指令
/memory
查看长期记忆
跨会话记住的信息，可查看和管理
/help
查看所有指令
当前版本全部可用指令的完整列表
§ 07 如何配置
没有图形设置界面，通过文件实现配置。.claude 文件夹默认隐藏，在文件资源管理器 → 查看 → 勾选「隐藏的项目」可显示。
C:\Users\你的用户名\.claude\
├─ CLAUDE.md  ← 全局指令
├─ settings.json ← 技术设置
└─ memory\     ← 长期记忆
~\.claude\CLAUDE.md
全局指令 · 最重要
对你打开的所有项目都生效。适合放普适规则，如：所有回答用中文、解释技术内容用通俗语言、修改文件前先列出改动计划。安装脚本已预置一份。
工作文件夹\CLAUDE.md
项目指令 · 按需设置
只对当前文件夹生效。适合放项目专属背景，如：这个文件夹是整理用户反馈的、不要动「已归档」目录。Claude 会同时读取全局指令和项目指令。
如何修改全局指令
启动 Claude Code 后直接说：

「帮我打开全局配置文件，我想加几条指令」

它会直接帮你写进去，不需要手动找文件。
§ 08 遇到问题？
弹出蓝色「Windows 保护了你的电脑」
点「更多信息」→「仍要运行」。这是 Windows SmartScreen 对未签名脚本的正常安全提示。
弹出「用户账户控制」确认窗口
点「是」。安装 Git 和 Node.js 需要管理员权限，这是 Windows 正常的安全确认流程。
安装中途卡住不动了
等待 5 分钟无反应，关闭窗口后重新双击运行安装脚本，已完成的步骤会自动跳过。
输入 claude 后提示「不是内部或外部命令」
确认已关闭旧窗口，重新打开新的 PowerShell 再试。如果仍不行，重启电脑后再试。
